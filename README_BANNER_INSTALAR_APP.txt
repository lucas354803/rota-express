ATUALIZAÇÃO - Banner para instalar app

Foi adicionado um banner fixo no rodapé com visual mais profissional para incentivar cliente/motoboy a instalar o PWA.

O banner aparece quando o navegador dispara o evento beforeinstallprompt.

Texto do banner:
📲 Instale o app Rota Express
Receba entregas em tempo real, notificações no celular e acompanhe tudo mais rápido.

Botões:
- 📥 Instalar agora
- Depois

Observação:
Se o usuário clicar em "Depois", o aviso fica escondido nesse celular/navegador.
Para aparecer novamente, limpe os dados do site ou o localStorage.
